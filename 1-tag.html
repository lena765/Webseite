Die erste Woche

zusammenfassung der ersten Woche:


Die erste Woche bei LeanIX begann am 09.03.2020. In den ersten drei Tagen haben wir eine Computersprache namens Python kennengelernt. Mit der Seite Code Combat lernt man kann man in kleinen Schritten das prigramieren lernen. Außerdem haben wir uns überlegt das es ganz cool wäre eine eigene Webseite zu erstellen in der wir über unser Praktikum berichten. Dazu haben wir erstmal gelernt was CSS, HTML, GIT, Github und Github Pages ist.
